HTML elements

Anchor link

  • Always link to HTTPS when possible.
  • Requires iui-anchor class name.
Bentley Systems  - A basic anchor link.
Download  - Forces the link to download.
Email  - Opens an email with filled out name, subject, and body.
Call  - Starts a call.

Abbreviation

  • Hover over an abbreviation to display the full term.
  • You will need to find an alternative method for touch devices.

Images

  • Include support for @2x at minimum, @3x at maximum.
  • Alt tag should describe what the picture depicts as if you were explaining it to a blind person.  Source .
  • Title tag should only be included if you need to display a tooltip.
  • Do  NOT  display SVGs within an IMG tag.
